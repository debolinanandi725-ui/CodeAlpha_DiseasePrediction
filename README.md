# 🩺 AI Disease Prediction System

A Machine Learning-based web application that predicts the possibility of diseases using patient medical data and generates a professional medical report.


---

## 📌 Project Overview

This project is developed as part of the **CodeAlpha Machine Learning Internship**.

The system uses multiple machine learning algorithms to analyze patient data such as age, symptoms, and medical test results to predict the likelihood of diseases.

---

## 🎯 Objective

To build a predictive model that can assist in early detection of diseases using structured medical datasets.

---

## 🧠 Machine Learning Models Used

- Logistic Regression  
- Random Forest  
- Support Vector Machine (SVM)  
- XGBoost  

---

## 📊 Features

✅ Predict disease based on user input  
✅ Clean and interactive UI using Streamlit  
✅ Multiple ML models for comparison  
✅ Generates downloadable PDF medical report  
✅ User-friendly interface for non-technical users  

---

## 🗂️ Dataset

Datasets used from UCI Machine Learning Repository:

- Heart Disease Dataset  
- Diabetes Dataset   

---

## ⚙️ Tech Stack

- Python  
- Streamlit  
- Scikit-learn  
- XGBoost  
- NumPy & Pandas  
- ReportLab (for PDF generation)
